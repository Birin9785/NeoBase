## TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install nodejs
> pkg install libwebp
> pkg install git
> pkg install ffmpeg
> pkg install yarn
> termux-setup-storage
> git clone https://github.com/Lexxy24/NeoBase.git
> cd NeoBase
> yarn
> rm -r session.json
> npm start
```
